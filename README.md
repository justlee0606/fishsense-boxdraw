# fishsense-boxdraw

### Instructions

1. Clone the repo
   ```sh
   git clone https://github.com/justlee0606/fishsense-boxdraw.git
   ```
2. Set up and activate virtual environment
   ```sh
   python -m venv env
   source env/bin/activate
   ```
3. Install packages
   ```sh
   pip install -r requirements.txt
   ```
4. Run boxdraw
   ```sh
   python main.py
   ```
